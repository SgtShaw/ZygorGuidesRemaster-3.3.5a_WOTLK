Place your guides in files Guide01.lua, Guide02.lua, ... in this directory, to have them auto-loaded on startup.

Optional leveling profile switch:
- Default (recommended): `Leveling\ZygorGuidesAlliance.lua` (TrinityCore-oriented).
- Optional fallback: `Leveling_Original\ZygorGuidesAlliance.lua` (original, unmodified Zygor source content).
- To use fallback, edit `Guides\Autoload.xml`: disable the default Alliance leveling script line and enable the `Leveling_Original` line.
